# Worldbuilding

## Setting Summary
- 完成原稿から読む限り、物語の主舞台はミルフィ村、ベルノア樹海、ひだまり盆地を軸に展開する。

## Spatial Notes
- 初期章のキーワード: 静かじゃなくなったミルフィ村と、樹海への引っ越し / ベルノア樹海入りと、ひだまり盆地発見 / 仮住まいと、はじめての伐採 / 水路と畑と、風呂ができるまで / 樹海探索と、埋もれた水路跡 / 工房開きと、元補給官バルド編 / 薬草園と、治療師ミレナ編 / 行商ペリックと、最初の交換市編

## Rules To Preserve
- 日常と開拓の積み上げを壊さない
- 樹海の神秘と共同体形成を両立させる
- 戦闘や事件よりも生活の具体と関係性の変化を重視する
